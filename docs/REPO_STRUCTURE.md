# Repo Structure

- `docs/` — public doctrine, compiler contract, proof-of-life, positioning, and system breakdown
- `specs/agi/` — public constitutional schemas, law docs, transitions, and example artifacts
- `specs/facts_plane/` — public facts-plane/control-kernel/governance schemas and examples
- `tools/fp_verify/` — public offline verifier package, examples, docs, and tests
- `examples/` — public-facing demo and walkthrough materials
- `references/` — supporting reference documents for buyer or reviewer use

This repository is intentionally narrow. It is designed to prove the public surface without disclosing the full internal runtime stack.
